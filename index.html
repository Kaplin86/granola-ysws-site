How does this work?


Create a breakout clone / brickbreaker in GODOT.
Publish your game to a hosting platform (like itch.io) for the world to see...
...and get free granola bars (and any other bar-shaped treats)!

Requirements

  • You are a high schooler (under 18)
  • Your game must be made in Godot.
  • The game must be published to a hosting platform (like itch.io).
  • The game must be a brickbreaker / breakout clone.
  • The game must be original, created for this YSWS specifically.
  • NO AI is allowed for code or imagery. Though, you can use AI to assist with the development process (like learning or coming up with ideas).
  • Your game must be open source.
